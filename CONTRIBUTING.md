# Contributing to ParSec Nova

Thank you for your interest in contributing! We want to make world creation as accessible as possible using AI assistance.

## 🌌 Creating New Worlds
You can contribute by designing new sectors. We use a standardized JSON format for world definitions.

### How to use AI for World Gen
We recommend using an LLM (like ChatGPT, Claude, or a local Llama model) with the following prompt structure:
> "Act as a level designer for ParSec Nova. Create a JSON sector definition using the following schema: [Link to your Schema-Doc]. Theme: [Your Theme, e.g., Asteroid Belt]."

3. AI-generated content must not violate the copyrights of existing franchises (e.g., do not use exact names/logos from Star Trek or Star Wars for public contributions).
